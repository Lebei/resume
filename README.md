    * 理解了组件化开发对于前端开发的重要性
* 项目总结
    * 在本项目中，我担任了前端开发一职，负责店铺类所有页面的编写，包括首页、列表页、产品详情页等。在这个项目中，充分认识到了前端架构的重要性。在随着多人开发同时进行页面越来越多的情况下，怎么合理把握好前端结构显得尤为重要。在开发的过程中，很多页面组件是通用的，因此怎么利用前端模块化来提高代码复用性成为一个很重要的任务。总的来说，这个项目实战型很强，收获很大。
